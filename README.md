# Warehouse Cloud Robotics 🚀

A ROS2 Humble based multi-robot warehouse automation system designed for intelligent task scheduling, crash recovery, and experiment logging.

## ✨ Features
- EDF-based task scheduling
- Multi-robot task allocation
- Crash recovery using checkpoints
- CSV experiment logging
- Custom ROS2 messages and services
- Cloud-deployment-ready architecture
- Docker-ready workspace
- Clean modular ROS2 package structure

## 🛠 Tech Stack
- ROS2 Humble
- Python
- Ubuntu WSL
- VS Code
- GitHub
- CMake
- Colcon
